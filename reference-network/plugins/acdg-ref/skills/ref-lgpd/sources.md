### Mapa de artigos-chave

| Artigo(s) | Tema |
|-----------|------|
| Art. 1°–3° | Âmbito de aplicação, definição de atividade de tratamento, âmbito territorial |
| Art. 4° | Hipóteses de exclusão (uso pessoal, jornalístico, segurança pública etc.) |
| Art. 5° | **Definições**: dado pessoal, dado pessoal sensível, dado anonimizado, titular, controlador, operador, encarregado, agentes de tratamento, consentimento, bloqueio, eliminação, transferência internacional |
| Art. 6° | **Princípios**: finalidade, adequação, necessidade, livre acesso, qualidade, transparência, segurança, prevenção, não discriminação, responsabilização |
| Art. 7° | Hipóteses de tratamento de dados pessoais (10 bases legais, incluindo consentimento e legítimo interesse) |
| Art. 8°–9° | Consentimento: forma, revogação, informação ao titular |
| Art. 10° | Legítimo interesse: condições e limites |
| Art. 11° | **Dados pessoais sensíveis** (inclui dados de saúde): hipóteses de tratamento mais restritivas |
| Art. 12° | **Anonimização**: dado anonimizado não é dado pessoal; critérios técnicos |
| Art. 13° | Pesquisa em saúde pública: dispensa de consentimento com requisitos |
| Art. 14° | Dados de crianças e adolescentes |
| Art. 15° | **Término do tratamento**: hipóteses e obrigações |
| Art. 16° | Conservação após término: hipóteses legais (cumprimento legal, auditoria, portabilidade, terceiros) |
| Art. 17°–22° | **Direitos do titular**: confirmação, acesso, correção, anonimização, portabilidade, eliminação, informação sobre compartilhamento, revogação |
| Art. 23°–30° | Tratamento pelo poder público |
| Art. 31°–35° | Transferência internacional de dados |
| Art. 37° | **Registro das operações** de tratamento (obrigatório para dados de saúde) |
| Art. 38° | Relatório de Impacto à Proteção de Dados Pessoais (RIPD) |
| Art. 39°–45° | Controlador e operador: responsabilidades, segurança, boas práticas |
| Art. 46°–51° | **Segurança**: medidas técnicas e administrativas; notificação de incidentes (Art. 48) |
| Art. 52°–54° | **Sanções**: advertência, multa (até 2% do faturamento, limite R$ 50 mi/infração), bloqueio, eliminação |
| Art. 55-A a 55-L | ANPD: estrutura, competências, processo administrativo |
